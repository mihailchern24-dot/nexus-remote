📚 API Reference

Authentication Register Peers Streaming Frames Wake-on-LAN

📚 Nexus Remote API

Base URL: https://nexus-remote.onrender.com

🔐 Authentication

POST /api/auth/register

Create a new account

{
  "email": "user@example.com",
  "password": "securePassword123"
}
POST /api/auth/login

Sign in to your account

{
  "email": "user@example.com",
  "password": "securePassword123"
}
→ {"token": "abc...", "peer_id": "nexus-abc123"}

📝 Register Device

POST /register

Register a device on the network

{
  "peer_id": "my-device-001",
  "platform": "windows",
  "compression": "zstd",
  "encryption": "aes_gcm"
}

👥 Peers

GET /peers

Get list of online peers

→ {"peers": ["device1", "device2"], "count": 2}

📺 Streaming

POST /start_stream

Start streaming between devices

{
  "source": "device1",
  "target": "device2",
  "quality": "high"
}
→ {"status": "streaming", "stream_id": "stream_..."}

📸 Frames

POST /send_frame

Send a video/audio frame

{
  "stream_id": "stream_...",
  "from": "device1",
  "target": "device2",
  "frame": "base64encodeddata",
  "type": "video"
}

⚡ Wake-on-LAN

POST /wol

Send Wake-on-LAN magic packet

{
  "mac": "AA:BB:CC:DD:EE:FF",
  "peer_id": "my-device"
}
→ {"status": "wol_sent", "success": true}
