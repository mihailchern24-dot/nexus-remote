Nexus Remote Documentation

Everything you need to know about using Nexus Remote

Getting Started

1
Create Account - Go to Register and sign up with email or Google/GitHub
2
Download App - Get the desktop app from Downloads
3
Run App - Execute python nexus_final_app.py or compile C++ version
4
Connect - Enter Peer ID of another device and click Connect

Website Features

Web Viewer

View remote screen directly in browser at /viewer. Enter Peer ID and click Connect.

Gamepad

Customizable mobile gamepad at /gamepad. Click Edit button to drag/resize buttons.

Speed Test

Test connection quality at /speedtest before streaming.

File Transfer

Drag & drop files at /files to send between devices.

Remote Support

Generate one-time access links at /support.

Dashboard

Manage devices, view stats at /dashboard (requires login).

Desktop App Features

Home Tab

Device info, Quick Connect, Share Screen, Start/Stop Capture

Devices Tab

Add devices with secret code, Wake-on-LAN, Sleep, View screen

Chat Tab

Real-time chat with connected device

Files Tab

Send and receive files

Security Tab

E2E encryption, MFA, Whitelist, Auto-lock settings

Stats Tab

Frames sent, data transferred, FPS

Wake-on-LAN

To wake a device remotely:

1
Add device in Devices tab with its MAC address
2
Click Wake button - Magic Packet sent to wake the PC
3
Wait 30-60 seconds for PC to boot
4
Click Connect to start streaming
